# 📋 Projeto: Listas Dinâmicas com JavaScript

Este projeto contém exemplos simples de manipulação do DOM utilizando **JavaScript puro**, com foco na criação dinâmica de listas HTML (`<ul>` e `<li>`).

---

## 🚀 Funcionalidades

O projeto possui três páginas com abordagens diferentes:

### 🔹 1. index.html
- Cria itens dinamicamente em uma lista
- Cada clique no botão adiciona um novo `<li>`
- Uso de `createElement` e `appendChild`

👉 Exemplo:
- Clique em **"Adicionar Item"**
- Um novo item aparece na lista

---

### 🔹 2. 2.html
- Cria uma lista com 5 itens de uma vez
- Utiliza `DocumentFragment` para melhor performance
- Evita múltiplas renderizações no DOM

👉 Destaque:
- Uso de `document.createDocumentFragment()`

---

### 🔹 3. 3.html
- Lista pré-definida de times brasileiros
- Adiciona todos os itens usando `forEach`
- Também utiliza `DocumentFragment`

👉 Times incluídos:
- Santos
- Flamengo
- São Paulo
- Palmeiras
- Corinthians
- Cruzeiro
- Inter
- Remo
- Bahia

---
